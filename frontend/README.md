# 🎨 DAF E-com Frontend Development

This directory contains the React-based frontend for the DAF E-com project.

## 🚀 Development

### Installation

```bash
npm install
```

### Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the application for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Previews the production build locally.

## 🛠️ Tech Stack Specifics

- **React 19**: Utilizing the latest React features and hooks.
- **Vite 8**: For ultra-fast development and optimized builds.
- **Tailwind CSS 4**: Modern CSS framework with native container queries and more.
- **Architecture**: Atomic design principles mixed with feature-based modularity.

## 📁 Key Directories

- `src/app`: Routing and global provider setup.
- `src/components`: UI components organized by layout and page sections.
- `src/features`: Logic and components grouped by domain (e.g., Blog, Services).
- `src/pages`: Main entry points for each route.
